# Stepan Grinchuk – 3D Modeler Portfolio

Портфолио 3D-моделера с интерактивными вращающимися моделями (glTF/GLB), переключением тёмной/светлой темы и анимацией появления карточек.

![Демонстрация портфолио](https://via.placeholder.com/800x400?text=3D+Portfolio+Preview)

## Возможности

- **3D-модели** – загрузка и автоматическое вращение моделей в формате `.gltf` или `.glb` (Three.js).
- **Ленивая загрузка** – модели подгружаются только когда карточка попадает в поле зрения.
- **Адаптивная сетка** – карточки подстраиваются под любые экраны.
- **Тёмная / светлая тема** – переключается кнопкой, выбор сохраняется в браузере.
- **Плавное появление** – карточки анимируются при скролле.
- **Чистая структура** – HTML, CSS и JS в отдельных файлах.

## Структура проекта
/
├── index.html # главная страница

│ ├── mrv1530.glb
│ └── ... (другие модели)
